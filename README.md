
Custom drone and flight control software based on ESP32 MCU


Components:
* Drone frame F450
* MPU6050 6-Axis Sensor
* 4x30A ESC
* 11.1V/3S 2200mAh 45C LiPO Upgrade Battery
* 4pcs 8x4.5 8045 CW CCW Propeller
* BMP280 pressure sensor
* 4xA2212 Brushless Motor 1000kv
* Flysky FS-i6X 6CH 2.4GHz RC Transmitter and Receiver FS-iA6B
