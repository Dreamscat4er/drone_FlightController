Values with |value| < threshold are returned as 0; others pass through unchanged.

Generic PID controller with error deadband and anti-windup.

Computes: out = Kp*e + Ki*∫e dt + Kd*de/dt, then constrains to [outMin, outMax]. If |e| < errDB, e is treated as 0; integral is slightly bled when e==0.

Parameters
targetDesired setpoint.
currentMeasured value.
integralReference to integrator accumulator (updated in-place).
prev_errorReference to previous cycle error (updated in-place).
KpProportional gain.
KiIntegral gain.
KdDerivative gain.
dtController timestep in seconds.
outMinMinimum output limit.
outMaxMaximum output limit.
errDBError deadband threshold.
Returns
Controller output after constraints.

Constrain a value and reset integral term on saturation (anti-windup).

If value exceeds [minVal, maxVal], the referenced integral is zeroed to prevent integral windup in controllers.

Parameters
valueValue to constrain.
integralReference to integrator accumulator (may be modified).
minValLower output limit.
maxValUpper output limit.
Returns
Constrained value within [minVal, maxVal].
